# 1. Subir o PostgreSQL

```bash
docker run -d \
  --name postgres-design-patterns \
  -e POSTGRES_PASSWORD=postgres \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_DB=postgres \
  -p 5432:5432 \
  -v $(pwd)/create.sql:/docker-entrypoint-initdb.d/create.sql \
  postgres:latest
```

# 2. Aguardar alguns segundos e executar o SQL

```bash
docker exec -i postgres-design-patterns psql -U postgres -d postgres < create.sql
```
